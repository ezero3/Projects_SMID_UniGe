# UniGe Data Science Projects 📊

Welcome to my UniGe Data Science Projects repository!

This repository collects the main projects developed during my Bachelor's Degree in Mathematical Statistics and Data Management.  
Its primary goal is to showcase my work, technical skills, and applied experience in data analysis, statistical modeling, and programming.

The projects cover a wide range of topics including statistics, machine learning, databases, and algorithm design, using tools such as R, Python, SAS, Java, and PostgreSQL.

---

## 📘 Projects Overview

### 🔹 Descriptive Statistics
**Olive Oil Classification (R)**  
Analyzed Italian olive oil samples to classify products based on fatty acid composition.  
Performed descriptive statistics, correlation analysis, and hierarchical clustering to identify geographical and chemical patterns across regions.

---

### 🔹 Applied Statistics

**Sampling Project (R)**  
Estimated total legal gambling expenditure in Liguria using:
- Simple random sampling  
- Stratified sampling (proportional & Neyman allocation)  
- Bootstrap resampling for variance estimation  

Focused on identifying optimal sampling strategies and comparing estimator efficiency.

**Time Series Project (R)**  
Analyzed monthly arrivals and departures in Australia (2014–2024) using the Box-Jenkins methodology.  
Built ARIMA/SARIMA models to capture trend and seasonality, and evaluated forecasts against post-COVID recovery dynamics.

---

### 🔹 Programming

**Java – Battleship Game Engine**  
Developed an advanced Battleship game featuring:
- Custom-shaped ships and mines  
- Player vs computer logic  
- Hit detection and game rules  
- Integration with a graphical interface  

**Python – Convex Hull Algorithm**  
Implemented an algorithm to compute the upper convex hull of planar points in linear time.  
Used stack-based techniques and geometric tests, ensuring **O(n)** worst-case complexity.

---

### 🔹 Databases

**UNIGE Social Sport (PostgreSQL)**  
Designed and implemented a relational database system including:
- ER modeling and schema design  
- Constraints, triggers, and views  
- Data population scripts and analytical queries  

Ensured normalization (BCNF), data integrity, and enforcement of business rules.

---

### 🔹 Statistical Programming (SAS)

**US Cars Dataset Analysis (1979)**  
Performed data analysis using SAS:
- Data cleaning and unit conversion  
- Correlation analysis and visualization (boxplots, scatterplots)  
- Comparative analysis across US, European, and Japanese cars  

Identified patterns in price, fuel efficiency, and regional manufacturing differences.

---

### 🔹 Machine Learning

**Algal Bloom Prediction (R)**  
Developed predictive models to classify hazardous algal blooms using environmental data.  
Techniques applied:
- Decision Trees, Random Forests  
- SVM, KNN, XGBoost  
- Class imbalance handling (ROSE)  

Evaluated models using AUC, accuracy, and sensitivity.  
Highlighted key environmental drivers of bloom risk.

---

### 🔹 Biostatistics

**Treatment Effect on PSA Levels (SAS)**  
Analyzed longitudinal data to evaluate treatment effects on PSA levels:
- Multiple linear regression on log-scale  
- Control for confounders (age, baseline PSA, seasonality)  
- Model diagnostics and assumption checks  

Estimated relative changes and interpreted clinical impact.

---

### 🔹 Linear Models

**Generalized Linear Models (SAS)**  
Modeled contraceptive use using binomial regression:
- Link functions: logit, probit, clog-log  
- Model comparison via AIC, BIC, deviance  

**Multivariate Models & Repeated Measures (SAS)**  
Analyzed heart rate response under different diets and exercise types:
- Multivariate linear models  
- Repeated measures ANOVA  
- Hypothesis testing using Wilks, Pillai, Hotelling-Lawley, Roy statistics
